# minute
minute is a tiny markdown parser.

## Installation
```
npm install minutemd
or
yarn add minutemd
```

## Usage
```typescript
import { convertToHTMLString } from 'minutemd'

const md = '# heading1';
convertToHTMLString(md)
// This outputs '<h1>heading1</h1>'
```
